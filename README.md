- 名字灵感来源于育碧游戏《看门狗》一代男主角——艾登·皮尔斯(Aiden Pearce)，也叫狗哥。
- 他是一个敢于对抗垄断势力(科技巨头布鲁姆公司)、捍卫公民隐私的人物。
